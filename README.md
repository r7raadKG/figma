# Metro Bonus Landing

В проекте собрана одностраничная посадочная, стилизованная под макет акции «Год с МЕТРО Бонус» на Vue 3 + Vuetify + TypeScript.

## Быстрый старт

1. Установите зависимости:
   ```bash
   npm install
   ```
2. Запустите локальный сервер разработки:
   ```bash
   npm run dev
   ```
3. Соберите production-версию:
   ```bash
   npm run build
   ```

## Структура
- `src/App.vue` — разметка и данные всех блоков страницы.
- `public/images` — фоновые иллюстрации и упрощённые логотипы партнёров, созданные на основе предоставленного макета.
- `src/styles/main.scss` — базовые стили и глобальные вспомогательные классы.
- `src/plugins/vuetify.ts` — инициализация темы Vuetify.
