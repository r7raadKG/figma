# Metro Bonus landing (Vue 3 + Vuetify)

Сборка повторяет мобильный лендинг «Год с METRO бонус». Стек: Vue 3, Vuetify 3, TypeScript и Vite. Все декоративные элементы и логотипы помещены в `src/assets`.

## Запуск

```bash
npm install
npm run dev
```

> Из-за офлайн среды зависимости не установлены. При наличии доступа к npm их можно подтянуть стандартной командой `npm install`.
